F1

F1 Data Hub

Professional Formula 1 data analytics — powered by FastF1 & Next.js

📊 Analytics Dashboard

Championship standings, race calendar, driver stats for seasons 2020–present.

📡 Telemetry Comparison

Compare speed, throttle, brake & gear traces between any two drivers.

🗺 Track Map

GPS racing line visualisation with speed heatmap and lap animation.

⏱ Qualifying Analysis

Sector breakdowns, fastest lap deltas and Q1/Q2/Q3 progression.

🤖 Race Prediction

AI win-probability model (RandomForest) trained on 2020–2024 seasons. Grid position, form, circuit history & more.

⚙ Quick Start

1.Start the backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
2.Start the frontend:
cd frontend
npm install
npm run dev
3.Open the dashboard at http://localhost:3000
AI Train the race prediction model (one-time, ~30 min):
cd backend
python scripts/build_dataset.py
python scripts/train_model.py
Downloads FastF1 data for 2020–2024 seasons and trains a RandomForest classifier. Restart the backend after training.
